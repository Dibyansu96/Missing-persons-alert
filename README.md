# Missing-persons-alert
FindThem is a comprehensive missing persons platform that helps reunite families through community collaboration, secure case reporting, advanced search, interactive maps, real-time alerts, and PWA support. Built to empower families, volunteers, NGOs, and law enforcement.
 Overview FindThem is a comprehensive web platform designed to help reunite missing persons with their families through community collaboration, advanced technology, and global reach. The platform serves as a centralized hub for reporting, searching, and sharing missing person cases worldwide.

🎯 Mission To leverage technology and community power to bring missing loved ones home safely and efficiently.
